# Каталог глоссариев Oghma

`manifest.json` содержит список наборов, которые приложение показывает в модальном окне на странице правил. Каждый набор — обычный экспорт `ogma.rules.export.v1`, поэтому его можно проверить и импортировать вручную.

## Обновление набора

Из корня проекта запустите:

```powershell
.\.venv\Scripts\python.exe scripts\build_glossary_pack.py --version 2026.07.22
```

Скрипт экспортирует текущий локальный глоссарий, пересчитает размер и SHA-256 и обновит запись в `manifest.json`. Версию набора нужно менять при каждой публикации содержимого.

Для нового набора укажите отдельные `--id`, `--filename`, `--title`, `--description` и `--version`. Идентификатор и имя файла должны состоять из строчных латинских букв, цифр и дефисов.

Публикуйте только материалы, на распространение которых у репозитория есть права.
