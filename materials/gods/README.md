# Каталог пантеонов Oghma

`manifest.json` описывает устанавливаемые наборы богов. Каждый файл набора совместим с форматом экспорта `ogma.gods.export.v1` и содержит только метки, относящиеся к выбранному сеттингу или пантеону.

Собрать каталог из полного экспорта богов:

```powershell
.\.venv\Scripts\python.exe scripts\build_god_catalog.py "C:\path\to\ogma-gods.json" --version 2026.07.22
```

Группы пантеонов задаются в `PACKS` внутри скрипта. Скрипт требует, чтобы каждый пантеон из исходного файла был явно отнесён к одному набору, создаёт стабильные ID, пересчитывает размеры и SHA-256 и затем обновляет манифест.

Публикуйте только материалы, на распространение которых у репозитория есть права.
